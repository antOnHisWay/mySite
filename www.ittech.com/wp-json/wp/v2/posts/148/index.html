What are Cookies?
\nCookies are data, stored in small text files, on your computer.<\/p>\n

When a web server has sent a web page to a browser, the connection is shut down, and the server forgets everything about the user.<\/p>\n

Cookies were invented to solve the problem “how to remember information about the user”:<\/p>\n

When a user visits a web page, his\/her name can be stored in a cookie.
\nNext time the user visits the page, the cookie “remembers” his\/her name.
\nCookies are saved in name-value pairs like:<\/p>\n

if ( isset( $_GET['key'] ) && isset( $_POST['key'] ) && $_GET['key'] !== $_POST['key'] ) {\r\n    wp_die( __( 'A key value mismatch has been detected. Please follow the link provided in your activation email.' ), __( 'An error occurred during the activation' ), 400 );\r\n} elseif ( ! empty( $_GET['key'] ) ) {\r\n    $key = $_GET['key'];\r\n} elseif ( ! empty( $_POST['key'] ) ) {\r\n    $key = $_POST['key'];\r\n}<\/pre>\n
 <\/p>\n
When a browser requests a web page from a server, cookies belonging to the page are added to the request. This way the server gets the necessary data to “remember” information about users.<\/p>\n
\n
Create a Cookie with JavaScript<\/h2>\n
JavaScript can create, read, and delete cookies with the\u00a0document.cookie<\/code>\u00a0property.<\/p>\n
With JavaScript, a cookie can be created like this:<\/p>\n
\n
document.cookie<\/span>\u00a0=\u00a0“username=John Doe”<\/span>;<\/span><\/div>\n<\/div>\n

\n
document.cookie<\/span>\u00a0=\u00a0“username=John Doe; expires=Thu, 18 Dec 2013 12:00:00 UTC”<\/span>;<\/span><\/div>\n<\/div>\n
